# FluidAgent Pro 项目简介

FluidAgent Pro 现在被定位为一套面向“工业视觉检测论文与实验闭环”的本地自动化控制器。

它的核心用途是帮助你把下面这条研究链路串起来：

`研究主线整理 -> 模型改进实现 -> 实验验证 -> 数据分析 -> 论文生成 -> Typst 导出`

## 当前适配的研究方向

- 金属表面缺陷检测
- 改进 YOLOv8
- 跨数据集实验设计
- 迁移学习与自建数据微调
- 工业部署评估
- 论文草稿与模板化导出

## 当前推荐的数据与任务

- 公开数据集：`NEU DET`、`GC10 DET`
- 自建数据集：`port_defect`
- 研究目标：兼顾缺陷检测精度、模型复杂度、推理速度与真实工业场景可部署性

## 它能做什么

- 读取 `research_plan.md`，按固定 Phase 拆分任务
- 调用 Codex 进行模型实现、训练脚本修订、日志检查、分析脚本编写与论文修订
- 调用 Gemini 根据实验结果生成论文草稿
- 在关键阶段暂停，等待人工审查后继续
- 支持从分析阶段回退到实现阶段，也支持从论文阶段回退到分析阶段

## 适用场景

- 你已经有一个 YOLOv8 或检测工程，希望把它组织成可持续迭代的论文工作流
- 你需要统一管理训练、验证、迁移学习、消融实验和部署评估
- 你希望最后得到一份可继续修改的 Typst 论文草稿与导出 PDF

## 核心输入

- `fluid_agent_pro.py`
- `research_plan.md`
- `metadata.json`
- `paper-template/clear-iclr/` 或后续新增的模板目录
- 你的检测工程代码、配置、日志与实验结果

## 主要特点

- CLI 与 GUI 共享同一套控制逻辑
- Phase 驱动，便于中断恢复和人工审查
- 更适合“改进 YOLOv8 + 多数据集实验 + 论文写作”的项目组织方式

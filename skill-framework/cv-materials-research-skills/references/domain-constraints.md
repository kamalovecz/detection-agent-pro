# 固定领域约束

## 研究域

本套 Skills 固定服务于以下研究域：

- 计算机视觉
- 材料缺陷检测

默认优先覆盖的工业视觉对象包括但不限于：

- 金属表面与钢板缺陷
- 焊缝、铸件、零部件表面缺陷
- 半导体、晶圆、PCB、封装缺陷
- 电池极片、复合材料、增材制造表面缺陷
- 纹理异常、微裂纹、腐蚀、孔洞、划痕、夹杂、压痕等

## 典型任务类型

- 分类
- 目标检测
- 语义分割 / 实例分割
- 异常检测 / 异常定位
- 严重度分级
- 实时部署评估

## 优先关注的问题

- 小样本与长尾类别
- 缺陷尺寸极小、形状细长、边界模糊
- 类间相似、类内差异大
- 光照变化、镜面反射、噪声、纹理干扰
- 标注成本高、标签不完备、开放集异常
- 跨产线、跨设备、跨数据集域偏移
- 实时性、算力、内存与工业部署约束
- 可解释性、稳定性与漏检/误检代价

## 常见方法族

- 检测：YOLO 系列、RT-DETR、DETR 变体、Faster R-CNN 系列
- 分割：U-Net 系列、DeepLab、Mask2Former 等
- 异常检测：PaDiM、PatchCore、Reverse Distillation、FastFlow、教师学生范式
- 表征学习：CNN、ViT、自监督预训练、轻量化骨干、工业纹理先验

## 评价指标

根据任务类型优先考虑：

- 检测：mAP、mAP50、mAP50-95、Precision、Recall、F1
- 分割：IoU、mIoU、Dice、Boundary F-score
- 异常检测：AUROC、AUPR、PRO、pixel-level AUROC
- 部署：FPS、Latency、Params、FLOPs、显存占用
- 泛化：跨数据集、跨工况、跨设备表现

## 复现与可比性要求

在涉及实验设计、结果分析或论文写作时，优先显式核对：

- 数据集划分方式
- 图像尺寸与预处理
- 数据增强
- 训练轮次、学习率、优化器、seed
- 推理阈值与后处理
- 硬件环境与版本
- 是否与基线使用相同训练预算

## 写作与评价边界

- 不要把问题泛化成无关的 NLP 或通用 LLM 讨论，除非它直接服务于材料缺陷检测研究流程。
- 不要脱离工业场景，只谈“通用 CV 性能提升”。
- 如果用户问题超出该固定领域，应先说明偏离点，再决定是否收缩回本领域。

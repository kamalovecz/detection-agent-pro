# 双语切换规则

## 默认规则

- 默认跟随用户当前使用的语言。
- 如果用户没有显式指定语言，则默认使用中文。
- 只有在用户明确要求时，才输出中英双语并列版本。

## 切换命令

- 切到中文：`/lang zh`、`切换中文`、`中文模式`
- 切到英文：`/lang en`、`switch english`、`english mode`

## 执行细则

- 切换命令对当前对话后续回复持续生效，直到用户再次切换。
- 中文模式下，数据集名、模型名、论文名、指标名保留原始英文写法；必要时在首次出现处补一句中文解释。
- 英文模式下，避免把中文术语生硬直译；优先使用社区常见说法。
- 如果用户的请求本身是中英混合，以命令优先；没有命令时，以提问主体语言优先。

## 输出模板同步

- 中文模式使用中文模板标题：
  - 任务理解
  - 当前问题诊断
  - 建议方案
  - 风险点
  - 下一步行动
  - 可交给 Codex 的实现任务
- 英文模式使用英文模板标题：
  - Task Understanding
  - Current Diagnosis
  - Recommended Plan
  - Risks
  - Next Actions
  - Codex Implementation Tasks
